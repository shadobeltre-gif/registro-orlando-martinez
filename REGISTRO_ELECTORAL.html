Cargar Padrón electoral

      

      

        Sube el padrón que te haya entregado el PRM o la JCE en cualquier formato: Excel (.xlsx/.xls), CSV o texto (.txt).
        El sistema busca automáticamente la columna de cédula (y nombre, si existe); si el archivo no tiene columnas claras,
        extrae cualquier número de 11 dígitos que encuentre. Los datos quedan guardados solo en este dispositivo.

Lotes Importar coordinadores desde Excel

      

        Columnas: Nombre (obligatoria), Celular y Correo (opcionales). El código de acceso y el
        código de evento se generan automáticamente para cada uno. Por el límite de seguridad de la nube, se crea
        aproximadamente 1 coordinador por minuto — para listas largas, súbelo desde Supabase → Authentication → Rate Limits.

Marca Datos de la campaña

      

        Cambia estos datos para adaptar el sistema a otro candidato. El nombre, cargo y circunscripción se actualizan
        en el login, el encabezado y el pie de página de la app en todos los dispositivos conectados.
      

      

      
      
      
      
      
      
      
      

        
        

          
          
        

      

      
      

        Para que otro candidato use el sistema completamente independiente de este (sin compartir sus votantes,
        coordinadores ni padrón), lo correcto es instalar esta misma app apuntando a un proyecto de Supabase distinto —
        igual que haces con tus otros clientes. Este formulario solo cambia el texto de esta instalación.
